## Current Project

### [Mainteer](https://www.mainteer.com.br)
> **Automotive Workshop Management Ecosystem**

O **Mainteer** é um ecossistema SaaS projetado para operações de oficinas mecânicas. A solução integra:

- Gestão de ordens de serviço  
- Controle de estoque com regras inteligentes  
- CRM orientado a performance  
- Fluxos automatizados e arquitetura escalável  

Tecnologias utilizadas: **React, Node.js, PHP, MySQL**, integrações REST e modelagem orientada a domínio.
